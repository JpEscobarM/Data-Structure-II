TDE: manipulação de arquivo binário

1) crie um arquivo de nomes de cidades (arqtexto, com pelo menos 100 cidades) no bloco de notas (notepad)

2) Faça um programa (em C) que leia os nomes desse arquivo (arquivo texto), e os escreva em um outro arquivo (arqbin) , em modo binário.

3) Faça uma função que leia um nome de cidade fornecido pelo usuário pesquise se ele está no arquivo arqbin (usando seek para pesquisar), informando o resultado da pesquisa  (está ou não no arquivo)

4) Faça uma função para inserir um novo nome de cidade (digitado pelo usuário) no arquivo arqbin.

5) Faça uma função para remover um  nome de cidade (digitado pelo usuário) no arquivo arqbin.
